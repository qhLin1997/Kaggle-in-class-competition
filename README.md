# Kaggle-in-class-competition

* requests
* numpy
* pandas
* xgboost
* seaborn
* matplotlib.pyplot
* statsmodels.api

Upload the folder Kaggle-in-class-competition to the jupyter nodebook, open the file model.ipynb, and run the cells in the file.
